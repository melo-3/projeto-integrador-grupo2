## Tema do Projeto
Página web sobre Receitas Brasileiras, apresentando 12 pratos típicos em formato de cards interativos.
# Objetivo
Desenvolver uma página web integrada a uma infraestrutura de rede simulada no Cisco Packet Tracer.
# Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Cisco Packet Tracer
# Estrutura da Rede
- LAN Clientes: 192.168.1.0/24
- Rede Servidor: 10.0.0.0/24
# Funcionalidades JavaScript
- Manipulação do DOM: geração dinâmica dos 12 cards de pratos via `script.js`
- Eventos: event listener de clique nos cards para abertura de modal com detalhes do prato
- Mudança de estilo dinâmica: adição/remoção da classe `ativo` no overlay para controlar exibição do modal
